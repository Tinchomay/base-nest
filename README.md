# Teslo API

1. Clonar proyecto

2. ```yarn install```

3. Crear el archivo .env a partir de .env.template

4. Levantar la BD con el comando
```
docker-compose up -d
```

5. Levantar: ```yarn start:dev```

6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
